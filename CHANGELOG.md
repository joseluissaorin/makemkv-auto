# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-20

### Added
- Initial release
- CLI tool with Typer framework
- Automatic disc detection and ripping
- Smart TV/Movie content detection
- Web UI dashboard with real-time updates
- Systemd service integration
- Beta key management
- Desktop notifications
- Comprehensive logging system
- Configuration management via YAML
- Multi-drive support architecture

### Features
- `makemkv-auto rip` - Manual disc ripping
- `makemkv-auto enable/disable` - Service control
- `makemkv-auto doctor` - Health checks
- Web UI at port 8766 with auto-refresh
- Progress tracking during rips
- Log viewer and system information
